# Git - Local

## Initializing Local Repository as git repository
```
Step1: Get the folder; right click to open git bash
Step2: git init
```
## Add a file
```
git add <filename>
```
## Add all files and folders
```
git add .
```
## Status
```
git status
```
## Commit all files 
```
git commit -m"commit changes"
```
## Log
```
git log
```
